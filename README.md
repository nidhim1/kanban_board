# Kanban Board

A drag-and-drop task management board built with Go, React, and Supabase.

## Tech Stack

- **Backend:** Go, Chi router
- **Frontend:** React 18, TypeScript, Tailwind CSS, dnd-kit
- **Database:** Supabase (PostgreSQL)
- **Auth:** Supabase Anonymous Auth with Row Level Security

## Project Structure

```
kanban-board/
├── backend/          # Go REST API
├── frontend/         # React + TypeScript app
├── supabase/         # Database schema and migrations
└── README.md
```
